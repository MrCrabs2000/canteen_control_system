Управление столовой
=

## Краткое описание проекта

Это веб-приложение для автоматизации работы школьной столовой с тремя уровнями доступа:

- Ученик - просмотр меню, оплата питания (разово или абонементом), указание пищевых особенностей, возможность оставлять отзывы на блюда. 
- Повар - формирование меню, контроль остатков продуктов и готовых блюд, создание заявок на закупку продуктов, просмотр отзывов учеников.
- Администратор - управление учётными0 записями пользователей, согласование заявок на закупку продуктов, просмотр статистики оплат и посещаемости, формирование отчётов о питании и затратам в Excel.

Система также включает модуль уведомлений для информирования всех участников о важных событиях, таких как изменение профиля, результаты согласования заявки, добавление отзыва учеником, истечение срока действия абонемента и другие.

Система реализована на **Flask** с использованием **SQLite**.

## Инструкция по установке


### Вариант 1: Запуск с помощью Docker

```bash
# 1. Установите Docker: 
https://docs.docker.com/get-docker/

# 2. Перейдите в папку проекта:
cd canteen_control_system

# 3. Запустите приложение одной командой:
docker-compose up --build

# 4. Откройте в браузере:
http://localhost:5000
```

### Вариант 2: Для разработки

#### 1. Клонирование репозитория

> git clone ```https://github.com/MrCrabs2000/canteen_control_system.git```

> cd canteen_control_system

#### 2. Создание виртуального окружения

Windows
> python -m venv venv
> 
> venv\Scripts\activate

Linux/Mac
> python3 -m venv venv
> 
> source venv/bin/activate

#### 3. Установка зависимостей

> pip install -r requirements.txt

#### 4. Запуск приложения

> python main.py

### Доступ к системе: 
Пароль администратора (логин: 'Admin') смотрите в консоли после запуска.
